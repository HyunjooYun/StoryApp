# KoPubWorld Dotum 폰트 설치 안내

이 폴더에 KoPubWorld Dotum 폰트 파일을 넣어주세요.

## 다운로드 방법
1. 네이버 검색: "KoPubWorld 다운로드" 또는 "KoPub 서체 다운로드"
2. 공식 사이트: https://www.kopus.org/biz/electronic/font.aspx
3. 또는 한국출판인회의 무료 서체 배포 페이지 방문

## 필요한 파일
- KoPubWorldDotumLight.ttf (Light, weight: 300)
- KoPubWorldDotumMedium.ttf (Medium, weight: 400)
- KoPubWorldDotumBold.ttf (Bold, weight: 700)

다운로드 후 이 폴더에 위 파일들을 넣고 `flutter pub get` 명령을 실행하세요.

## 임시 방안
폰트 파일이 없어도 앱은 실행되지만, 시스템 기본 폰트가 사용됩니다.
Medium 파일만 있어도 동작하며, 파일명을 위와 같이 맞춰주세요.
